# Changelog

## v0.1.0 — 2026-03-29

- Initial project scaffold with Astro 6.1.1, TypeScript 6 (strict), React 19, Tailwind CSS v4
- MDX content collections for blog posts
- Framer Motion fade-in animations via React islands
- Three analytics providers (Plausible, Vercel Analytics, Google Analytics) toggleable via env vars
- Minimal site structure: Hero, Experience timeline, Contact
- Base layout with Google Fonts (Inter + JetBrains Mono), dark theme
